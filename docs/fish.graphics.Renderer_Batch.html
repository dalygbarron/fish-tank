add(src, dst, scale)

Source:
  • graphics.js, line 518
Adds a sprite to the list of those to draw. I guess rotating would be good but I would have to do it in software and I dunno what the performance would be like.
Parameters:
Name Type Description
src fish.util.Rect is the src rectangle from the texture.
dst fish.util.Rect | fish.util.Vector is where to draw it on the screen. If it's a vector then that is the centre.
scale number is used to scale the sprite if you used a vector. If you used a rect it does nothing.
