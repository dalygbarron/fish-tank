Namespace: audio

fish.audio

This file provides audio playing and loading functionality and a basic sound player class. This player only supports playing audio files that are fully loaded into memory, there is no audio streaming because it would lag and suck. If you need more flexible audio playing then feel free to create your own class that does what you need.
