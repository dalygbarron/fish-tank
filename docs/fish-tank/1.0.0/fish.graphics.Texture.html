Class: Texture

fish.graphics.Texture(glTexture, width, height)

new Texture(glTexture, width, height)

Creates a texture object out of a gl texture. You probably don't want to instantiate one of these directly unless you are creating your own graphics system.
Parameters:
Name Type Description
glTexture number is the open gl reference to the texture.
width number is the width of the texture.
height number is the height of the texture.
