Class: Transition

fish.screen.Transition(pop, screennullable, messagenullable)

Represents a transition between screens on the screen stack.

Constructor

new Transition(pop, screennullable, messagenullable)

There are three different configurations that this constructor allows. When pop is true the current screen is removed, when screen is not null then that screen is placed on the stack. Thus, you can push a screen on this screen, replace this screen with another, or you can just pop this screen. If you set pop to false and screen to null then nothing will happen.
Parameters:
Name Type Attributes Default Description
pop boolean whether to pop the returning screen from the screen stack.
screen fish.screen.Screen <nullable>
 null is a screen to add to the screen stack if given.
message Object <nullable>
 null a message that will be given to whatever screen is going to next have reveal called on it.
