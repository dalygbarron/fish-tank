Class: LoadScreen

fish.screen.LoadScreen(graphics, after)

new LoadScreen(graphics, after)

Creates a loading screen that waits for a bunch of promises to evaluate.
Parameters:
Name Type Description
graphics is the game graphics object used to render stuff.
after is a function called with all the evaluated promises which should itself evaluate to a replacement screen.
...promises is all the promises.
