Namespace: gui

fish.gui

This file provides a kinda basic gui system for the user to interact with. It only uses button input by default but it should be able to do menu type stuff as well as game dialogue and basic hud if need be etc. In the future I might add mouse support to the default input system in which case I will also make the gui be able to use mouse at least if you want it to.
