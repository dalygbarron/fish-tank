Interface: Context

fish.screen.Context

Stores all the game's subsystems. Now, you will notice that these are all interface types that the engine provides. You will control what the implementing type is and for god's sake don't try to do any static type crazy bullshit with this. Just accept that the actual implementing types of these are the ones you asked for in your game. These interfaces are just the basic amount of functionality that the engine requires from each subsystem.
