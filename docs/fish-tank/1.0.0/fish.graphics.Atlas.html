getPatch(name, borderopt) → {fish.graphics.Patch}

Gets a 9-patch out of the atlas and makes it for you. If you pass the border argument then it is used to create the patch, but if you leave it as 0 then it tries to use the name to discern the border size of the patch by looking for a number at the end of the name. If neither of those things are present then an error will be thrown.
Parameters:
Name Type Attributes Default Description
name string name of the sprite the patch is made of.
border number <optional>
 0 the width of the borders of the patch.
