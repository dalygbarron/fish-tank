InputHandler

fish.input. InputHandler

new InputHandler(keymapopt, thresholdopt)

Source:
  • input.js, line 68
Implements:
  • fish.input.UiInput
An input handler system that unifies all input from gamepads / keyboard into one abstract input which is supposed to work like a gamepad basically. It only works with 1 player games for that reason.
Parameters:
Name Type Attributes Default Description
keymap Object.<string, string> <optional>
 {} a mapping from html key names to button on the virtual controller.
threshold number <optional>
 0.9 the threshold beyond which a gamepad axis is considered pressed.
