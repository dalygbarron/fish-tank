Class: BasicInput

fish.input.BasicInput(keymap, is)

new BasicInput(keymap, is)

An input handler that unifies all input from gamepads / keyboard into one abstract input which is supposed to work like a gamepad basically. It only works with 1 player games for that reason.
Parameters:
Name Type Description
keymap Object.<string, string> is a mapping from html key names to button on the virtual controller.
is number the threshold beyond which a gamepad axis is considered pressed.
